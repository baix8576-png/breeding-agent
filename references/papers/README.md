# papers

- Store paper-level evidence notes that support blueprint decisions and interpretation boundaries.

- `paper_summary_*.md`: concise paper summaries with methods, population/species, and applicability notes.
- `evidence_extract_*.md`: extracted claims mapped to downstream blueprint scopes.
