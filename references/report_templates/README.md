# report_templates

Planned documentation set for end-user result packaging.

Examples:
- QC summary template
- structure analysis summary template
- genomic prediction report template
- risk and caveat statement template

Current placeholder files:
- `qc-report-template.md`
- `structure-summary-template.md`
- `genomic-prediction-template.md`

Usage:
- These templates are for output packaging and reviewer communication.
- They are not substitutes for validated analysis results.
