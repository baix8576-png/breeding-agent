# docs

This directory stores project-level design notes, release notes, and architecture supplements
that do not belong in `README.md`, `AGENTS.md`, or repo-scoped skill definitions.

Key files:

- `HANDOFF.md`: mandatory continuity ledger with project-specific `HANDOFF v2` schema and resume checklist.
- `v1_5_system_map.md`: V1.5 executable-loop map and architecture summary.
- `genomic_agent_architecture_summary.md`: archived early architecture summary kept for historical context.
